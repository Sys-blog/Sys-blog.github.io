VSCode常用快捷键

Visual Studio Code（简称VSCode）是一款非常流行的代码编辑器，它提供了许多快捷键来提高开发效率。以下是一些常用的快捷键：

基本编辑

  • Ctrl + C - 复制选中的内容
  • Ctrl + V - 粘贴复制的内容
  • Ctrl + X - 剪切选中的内容
  • Ctrl + Z - 撤销上一步操作
  • Ctrl + Y - 重做上一步操作

导航

  • Ctrl + P - 快速打开文件
  • Ctrl + Shift + O - 跳转到符号
  • Ctrl + G - 跳转到指定行
  • Ctrl + Tab - 在打开的文件之间切换

多光标和选择

  • Alt + Click - 添加新的光标
  • Ctrl + Alt + ↑/↓ - 在上方/下方添加光标
  • Ctrl + D - 选择下一个匹配项
  • Ctrl + Shift + L - 选择所有匹配项

代码格式化和重构

  • Ctrl + Shift + I - 格式化选中的代码
  • Ctrl + . - 快速修复
  • F2 - 重命名符号

终端

  • Ctrl + ` - 显示/隐藏终端
  • Ctrl + Shift + ` - 创建新的终端

掌握这些快捷键可以显著提高你在VSCode中的编码效率。希望这些快捷键对你有所帮助！
