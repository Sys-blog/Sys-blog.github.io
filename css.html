CSS入门

CSS（层叠样式表）是一种用于描述HTML文档外观和格式的样式表语言。通过CSS，你可以控制HTML元素的颜色、字体、布局等。

基本语法

CSS规则由选择器和声明块组成。选择器用于选择要应用样式的HTML元素，声明块包含一个或多个属性和值。

选择器 {
    属性: 值;
    属性: 值;
}

选择器

选择器用于选择HTML元素。常见的选择器包括元素选择器、类选择器和ID选择器。

/* 元素选择器 */
p {
    color: blue;
}

/* 类选择器 */
.highlight {
    background-color: yellow;
}

/* ID选择器 */
#header {
    font-size: 24px;
}

常用属性

以下是一些常用的CSS属性：

  • color: 设置文本颜色
  • background-color: 设置背景颜色
  • font-size: 设置字体大小
  • margin: 设置外边距
  • padding: 设置内边距

示例

以下是一个简单的CSS示例，展示了如何设置文本颜色和背景颜色：

<style>
    .example-text {
        color: white;
        background-color: #4CAF50;
        padding: 10px;
    }
</style>

<div class="example-text">
    这是一个带有白色文本和绿色背景的示例。
</div>
